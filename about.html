saya adalah seorang developer, status saya masih pemula dibidang ini, tetapi saya dari kecil memang sangat ingin sekali menguasai komputer dan juga ilmu komputer lainnya, saya tertarik di bidang pemprograman ini karena saya mempunyai impian atau mendapat inspirasi dan berbagai hal positif dan juga karena perkembangan teknologi saat ini membantu saya untuk mengetahui keadaan teknologi kita saat ini yang disebut era nya digital. oleh karena itu saya mempelajari nya dari beberapa guru pembimbing saya dan alhamdulillah nya saya bisa meski masih ada beberapa kesalahan. banyak orang bilang belajar coding itu sia sia atau susah, tetapi disisi lain belajar coding itu sangat keren, menurut saya coding itu keren dan juga menguji logika kita serta keterampilan kita mengedit,membuat ataupun ide ide yang kita punya untuk membangun suatu website atau lainnya. impian saya yaitu ingin menguasai segala hal tentang komputer baik itu hardware ataupun software nya dan saya akan terus berusaha untuk menggapai impian saya saat ini untuk bisa menjadi seorang developer. dan satu lagi impian saya adalah ingin membuat game bautan saya sendiri yang bermanfaat ataupun fun dimainkan oleh banyak orang. sekian detail dari data diri saya, semoga dari data diri saya ini bisa memacu semangat kalian yang ingin terjun ke dunia percodingan dan jangan mudah menyerah ataupun mengeluh hadapilah itu kenyataannya memang tidak semua orang mudah atau cepat memahami suatu hal, tetapi tidak ada orang yang tidak bisa sukses ataupun menguasai hanya saja dari diri kita yang kalah tekad kita dalam belajar. semangat untuk kalian yang ingin menjadi seorang pemprograman salam dari saya terima kasih :) .
